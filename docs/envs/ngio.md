# Ngio

ngio is a Python library for working with next-generation image formats, providing efficient access to modern image storage formats. Learn more about ngio at [ngio repository](https://github.com/fractal-analytics-platform/ngio).

## Environment

ngio is available in the `ngio` environment.

To activate the environment, pre-fix your commands with:
```bash
pixi run -e ngio YOUR_COMMAND
```
