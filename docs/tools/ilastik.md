# ilastik

ilastik is an interactive machine learning toolkit for image classification and segmentation without requiring machine learning expertise. Learn more about ilastik at [ilastik.org](https://www.ilastik.org/).

## Environment

ilastik is available in the `ilastik` environment.

To activate the environment, pre-fix your commands with:
```bash
pixi run -e ilastik YOUR_COMMAND
```

## Available Tasks

### Start ilastik GUI
Launch the ilastik graphical user interface:
```bash
pixi run ilastik
```
