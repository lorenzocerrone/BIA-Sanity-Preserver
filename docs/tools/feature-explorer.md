# Feature-explorer

Feature Explorer is a Python dashboard application for exploring OME-Zarr images alongside tabular data. Learn more about Feature Explorer at [feature-explorer repository](https://github.com/fractal-analytics-platform/feature-explorer).

## Environment

Feature Explorer is available in the `feature-explorer` environment.

To activate the environment, pre-fix your commands with:
```bash
pixi run -e feature-explorer YOUR_COMMAND
```

## Available Tasks

### Start Feature Explorer Dashboard
Launch the interactive Feature Explorer dashboard:
```bash
pixi run feature-explorer
```
