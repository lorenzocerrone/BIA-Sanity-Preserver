# Plant-seg

PlantSeg is a deep learning-based tool for cell segmentation in plant tissues. Learn more about PlantSeg at [PlantSeg repository](https://github.com/PlantSeg/PlantSeg).

## Environment

PlantSeg is available in the `plantseg` environment.

To activate the environment, pre-fix your commands with:
```bash
pixi run -e plantseg YOUR_COMMAND
```

## Available Tasks

### Start PlantSeg GUI
Launch the PlantSeg graphical user interface:
```bash
pixi run plantseg-gui
```
