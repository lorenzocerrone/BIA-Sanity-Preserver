# Napari (OME-Zarr)

Napari is a fast, interactive, multi-dimensional image viewer for Python with OME-Zarr support for handling large datasets. Learn more about Napari at [napari.org](https://napari.org/).

## Environment

Napari with OME-Zarr support is available in the `napari-ome-zarr` environment.

To activate the environment, pre-fix your commands with:
```bash
pixi run -e napari-ome-zarr YOUR_COMMAND
```

## Available Tasks

### Start Napari
Launch the napari viewer with OME-Zarr support:
```bash
pixi run napari
```
